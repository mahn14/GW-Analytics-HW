Intro

The goal of this project was to create a regression model to predict the SalePrice variable of over a thousand houses in Ames, Iowa. Our strategy revolved around using mainly linear regression, evaluated using root-mean-squared-error. This process was divided into several steps.

Data preprocessing. This step focuses on readying the data even before exploring it in depth. While the data set was provided in a neat .csv file, there were other problems such as missing values or skewed distributions.

Data exploration. The next step was to examine the variables and their relationships. A deeper understanding the variables allows for a better model.

Data processing. After consolidating our final variables, fixing them to our liking, and understanding their relationships, it is time to prepare them specifically for the models we will be using. While some of the parts in this step is almost identical to preprocessing, now we are focusing on enhancing our varialbes rather than fixing.

Model Building. This can be broken into many steps. The first would be in actually training our models themselves by finding optimal hyperparameters to be used for our final model. We chose a gridsearch algorithm using 5-folds.

Model Validation. Now that we have our final model, we must make sure that it is viable for generalizations. We test our model using data that we had set aside for this purpose.

Summary It is always a good idea to take validation a step further and look once again at the big picture. Did we meet our initial goals? If not, what has changed since then? We must determine at this point whether to close the project or start again at a previous step.
