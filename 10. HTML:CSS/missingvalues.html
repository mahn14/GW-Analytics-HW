Missing Values

Another necessary step before model building is dealing with missing values.

While many model building steps are done to help create a better model, this step is crucial in that the model literally cannot be built without it. If there are missing values in the data, then many algorithms will simply not work. Some variations of methods to fix this problem are simple, though. We chose to get rid of many of the variables since some of them were simply missing too many values. For others, we filled in the missing values using mean, median, mode, or unique imputations.
